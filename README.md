# ⌁ IPSW Archive

> Official Apple firmware catalog — clean, fast, no fluff.

Browse and download IPSW files for every Apple device. Built with React + Vite, powered by the [IPSW.me API](https://api.ipsw.me).

## ✨ Features

- 🔍 **Search** — find devices by name or identifier
- 📱 **All devices** — iPhone, iPad, Mac, Watch, TV, iPod, HomePod
- ✅ **Signing status** — see which firmwares are still being signed
- 🎨 **Clean UI** — no ads, no clutter
- ⚡ **Fast** — built with Vite, optimized for speed

## 🚀 Quick Start

```bash
# Clone
git clone https://github.com/Lumid-Off/iPSWDownloader.git
cd iPSWDownloader

# Install
npm install

# Run
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## 📦 Build

```bash
npm run build
npm run preview
```

## 🛠 Tech Stack
